About NewsRAG

An intelligent Serverless Retrieval-Augmented Generation (RAG) agent that parses, indexes, and synthesizes live news in real-time.

Architectural Diagram

How the Project is Built

📥

1. Real-time Ingestion

EventBridge cron rule triggers an AWS Lambda function daily to ingest news RSS feeds, articles, and updates, storing raw files in an Amazon S3 Data Lake.

🧠

2. Embedding & Vector Indexing

Amazon Bedrock Titan Embedding model embeds documents, indexing them with a lightweight FAISS (Facebook AI Similarity Search) index for semantic retrieval.

⚡

3. Synthesis & Caching

Amazon Bedrock Nova Lite generates natural language answers based on retrieved context, with DynamoDB caching responses to keep execution latency low.
